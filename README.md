# BiosUP Catalog

This repository stores only official BIOS lookup rules/templates for BiosUP.

It does **not** store BIOS files, firmware binaries, cracks, mirrors or third-party downloads.
The application uses this catalog to find the correct official support page for a detected motherboard, then downloads BIOS only from official vendor domains.

Current approach: exact hand-curated model aliases + generic vendor templates + local fallback copy bundled with the app.
