# Catalog format

`catalog.json` is the index file. It lists vendor files in `vendors/`.

Each vendor file should describe:

- `manufacturer`
- `aliases`
- `officialDomains`
- `genericRules`
- `models`

Recommended model fields:

- `model`
- `aliases`
- `supportSlug`
- `supportPageUrl`
- `boardId`
- `revision`
- `revisionRequired`
- `safeWithoutRevision`
- `expectedBiosFilePattern`
- `confidence`
- `lastChecked`
- `source`
- `latestBios`
- `verifiedLatestBios`

Confidence levels:

- `template`: generated by vendor URL template
- `manual`: added by hand
- `page_verified`: support page opened and matched
- `file_verified`: BIOS archive was downloaded and checked by the app

The catalog helps the app find official support pages. The app still verifies the downloaded file before marking anything safe.
