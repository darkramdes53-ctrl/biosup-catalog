# Future roadmap

Good future additions:

- More verified model aliases.
- Vendor-specific validation notes.
- Automated official page availability checks.
- Separate verified catalog releases.
- App-side support for signed catalog releases.

Avoid:

- Storing BIOS files.
- Using third-party mirrors.
- Keeping free proxy lists in the repository.
- Marking catalog entries as safe without file verification in the app.
